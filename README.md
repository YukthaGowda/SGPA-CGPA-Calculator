# 🎓 SGPA & CGPA Calculator

A web-based calculator that helps students compute their SGPA (semester GPA) or CGPA (cumulative GPA) — built as a team project, with dynamic subject/semester entry rather than a fixed form.

## What it does

- Lets the user choose **SGPA** (single semester) or **CGPA** (cumulative across semesters)
- Dynamically generates input fields based on the number of subjects/semesters entered — no hardcoded row limit
- Calculates and displays the result instantly in-browser, no backend or page reload required

## Stack

`HTML5` · `CSS3` · `JavaScript` (vanilla, DOM manipulation)

## Run it

```bash
git clone https://github.com/YukthaGowda/Team-Supremes.git
cd Team-Supremes
open index.html
```

No build step or dependencies — it's a static client-side app.
